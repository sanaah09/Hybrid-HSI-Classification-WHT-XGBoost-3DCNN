# Hybrid Framework for Efficient Hyperspectral Image Classification

## Overview

This repository contains the implementation of a hybrid framework for hyperspectral image classification using:

- Walsh-Hadamard Transform (WHT)
- XGBoost Band Ranking
- K-Means Clustering
- 3D CNN Classification
