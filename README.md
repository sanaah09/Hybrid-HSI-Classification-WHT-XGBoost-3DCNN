# Hybrid Framework for Efficient Hyperspectral Image Classification

## IEEE Conference Project

Authors:
- Sanjana S
- Navina Ingersol
- Vishnu Priya R
- Shanmuga Sundari P

## Overview

This project presents a hybrid hyperspectral image classification framework using:

- Walsh-Hadamard Transform (WHT)
- XGBoost Band Ranking
- K-Means Clustering
- 3D CNN Classification

## Datasets

- Indian Pines
- Pavia University

## Project Files

- Indian_Pines.py
- Pavia_University.py
- IEEE Conference Paper

## Results

| Dataset | Accuracy |
|----------|----------|
| Indian Pines | 97.27% |
| Pavia University | 99.64% |
